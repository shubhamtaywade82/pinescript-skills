# Built-ins

Pine Script® contains hundreds of built-in variables and functions.
- **Variables**: `close`, `open`, `volume`, `bar_index`, `time`, `syminfo.ticker`.
- **Technical Analysis**: `ta.sma()`, `ta.ema()`, `ta.rsi()`, `ta.macd()`.
- **Math**: `math.abs()`, `math.log()`, `math.max()`.

See the [Built-in Reference Manual](../../builtins-reference.md) for a complete list of built-in variables and functions.
