# Errors and Warnings

Pine Script® uses runtime errors, compilation errors, and compiler warnings to help prevent unintended or erroneous script behaviors.

### Contents

- **[Overview](overview.md)**: Introduction to the types of errors and compiler warnings in Pine Script®.
- **[CE10101](CE10101.md)**: Compilation error: Condition must evaluate to a boolean.
- **[CW10003](CW10003.md)**: Compiler warning: Functions relying on history should be called on each calculation.
- **[RE10139](RE10139.md)**: Runtime error: Memory limits exceeded.
- **[RE10143](RE10143.md)**: Runtime error: Requested historical offset is beyond the buffer limit.
