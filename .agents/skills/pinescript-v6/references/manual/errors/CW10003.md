# CW10003: Function Should Be Called on Each Calculation

### Message
`The function "X" should be called on each calculation for consistency. It is recommended to extract the call from this scope.`

### Cause
This warning occurs when you place a built-in function that relies on historical calculations (such as `ta.sma()`, `ta.ema()`, or `ta.change()`) inside a conditional block (`if` or `switch`) or a loop. Because the block does not execute on every bar, the historical buffer for the function becomes inconsistent, resulting in incorrect calculation results.

### ❌ Problematic Code
```pinescript
//@version=6
indicator("CW10003 Demo")

bool isRaining = close > open
float runningAverage = 0.0

// ❌ Warning: ta.sma() is inside conditional block
if isRaining
    runningAverage := ta.sma(close, 14)
```

### ✅ Solution
Move the historical function call to the global scope so it calculates on every single bar, then assign its output inside the conditional block.

```pinescript
//@version=6
indicator("CW10003 Solution")

bool isRaining = close > open
float runningAverage = 0.0

// ✅ Correct: Calculated on every bar
float calculatedSma = ta.sma(close, 14)

if isRaining
    runningAverage := calculatedSma
```
