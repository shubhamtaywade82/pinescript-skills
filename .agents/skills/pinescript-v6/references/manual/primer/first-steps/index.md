# First Steps

To start writing scripts in Pine Script® v6, you will need to open the Pine Editor on TradingView.

### opening the Pine Editor
1. Open any chart on TradingView.
2. Click on the **Pine Editor** tab at the bottom of the screen.
3. Use the **Open** menu to create a new script or start from a blank indicator template.

### Adding a Script to the Chart
- Once your code is ready, click **Save** to save the script to your account.
- Click **Add to chart** to run the script. The script's output will display either overlaid on the main price panel or in a separate pane underneath.
