# Next Steps

After creating your first indicator, you can expand your scripts in several directions:

### 1. Indicators vs. Strategies
While **Indicators** (`indicator()`) plot analysis, **Strategies** (`strategy()`) allow you to backtest trading rules by placing orders using functions like `strategy.entry()` and `strategy.close()`.

### 2. Libraries
If you write utility functions that you want to share across multiple scripts, you can write a **Library** (`library()`) to organize and publish reusable Pine code.
