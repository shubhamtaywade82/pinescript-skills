# Identifiers

Identifiers are names used for user-defined variables, functions, types, and enums.

### Naming Rules
- Must start with an uppercase or lowercase letter, or an underscore (`_`).
- Can contain letters, digits, and underscores.
- Are case-sensitive (e.g., `myVar` and `myvar` are different).
- Cannot match Pine Script® reserved keywords (like `if`, `var`, `indicator`).

### Style Guide Suggestion
- Use camelCase for variables, functions, and types.
- Use uppercase SNAKE_CASE for constants.
