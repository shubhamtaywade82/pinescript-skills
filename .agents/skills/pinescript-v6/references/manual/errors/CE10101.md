# CE10101: Condition Must Evaluate to Boolean

### Message
`The condition of the "X" statement must evaluate to a "bool" value.`

### Cause
In Pine Script® v6, expression conditions in conditional structures (`if` and `switch` statements) must strictly evaluate to a `bool` type (`true` or `false`). Implicit type coercion from numbers or strings to booleans is not allowed.

### ❌ Problematic Code
```pinescript
//@version=6
indicator("CE10101 Demo")

int count = 5
// ❌ ERROR: count is an integer, cannot be implicitly checked
if count
    log.info("Count is non-zero")
```

### ✅ Solution
Use an explicit comparison operator to produce a boolean result.

```pinescript
//@version=6
indicator("CE10101 Solution")

int count = 5
// ✅ Correct: Explicit boolean comparison
if count != 0
    log.info("Count is non-zero")
```
