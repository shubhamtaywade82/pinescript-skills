# Visuals

Visuals in Pine Script® control what is displayed on the chart pane. This includes plots, fills, backgrounds, custom bar colors, drawings, and tables.

- **[Backgrounds](backgrounds.md)**: Changing chart backgrounds with `bgcolor()`.
- **[Bar Coloring](bar-coloring.md)**: Changing bar colors with `barcolor()`.
- **[Bar Plotting](bar-plotting.md)**: Plotting custom bars or candles with `plotbar()` and `plotcandle()`.
- **[Colors](colors.md)**: Working with the `color` type, RGBA transparency, and palette functions.
- **[Fills](fills.md)**: Shading spaces between plots or lines using `fill()`.
- **[Levels](levels.md)**: Rendering horizontal reference levels with `hline()`.
- **[Lines and Boxes](lines-and-boxes.md)**: Drawing lines, boxes, and polylines on the chart.
- **[Plots](plots.md)**: Standard plot outputs using `plot()`.
- **[Tables](tables.md)**: Creating customized multi-cell layouts and dashboards.
- **[Text and Shapes](text-and-shapes.md)**: Rendering text, shapes, and markers using `plotshape()`, `plotchar()`, and `label.new()`.
